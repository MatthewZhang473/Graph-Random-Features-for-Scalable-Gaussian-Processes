# Social network datasets

We use the Stanford Large Network Dataset Collection for the social network datasets.

To access them, visit: https://snap.stanford.edu/data/

The datasets used are:

- Twitch gamers: https://snap.stanford.edu/data/twitch_gamers.html
- Facebook: https://snap.stanford.edu/data/facebook-large-page-page-network.html
- Youtube: https://snap.stanford.edu/data/com-Youtube.html
- Enron: https://snap.stanford.edu/data/email-Enron.html
